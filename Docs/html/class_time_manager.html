Detailed Description

Execute unity event after specific second.

Tips: attach this component to gameObject and assign functions to countdownEndEvents. Call StartGame() to execute events after specific second. TimeManager normally use to execute event once after specific second. For example, call RestartScene() in SceneSwitch to restart scene, after 120 seconds in Registration Page
