Detailed Description

Add options to TMP_Dropdown field from a file Tips: put a text file with line by line text into assets folder, attach this component to a TMP_Dropdown gameObject.

You can add the options in editor by right click on this component in inspector and select Feed Notes: The options will automatically feed into dropdown field in Start()
