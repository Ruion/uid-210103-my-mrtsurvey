Detailed Description

Visual a slider value to a TextMeshProUGUI component By default it only execute events once and disable itself.

Change this behaviour by toggling executeOnce boolean. Tips: Attach SwipeDetector to a gameObject and drag function in to OnSwipeLeft / OnSwipeRight event.
