Detailed Description

Execute unity event with delay.

Each event has its own delay, set delay to 0 if no delay desired. Tips: attach this component to gameObject and assign functions to eventsOnAwake. Call Run() to execute events. EventSequencer can be use in many case like play sound after delay, looping music after delays, gather and execute function in sequences.
