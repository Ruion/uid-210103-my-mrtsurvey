Detailed Description

Execute Unity Event whenever left or right swipe is detected.

Events for left swipe and right swipe are separated By default it only execute events once and disable itself. Change this behaviour by toggling executeOnce boolean. Tips: Attach SwipeDetector to a gameObject and drag function in to OnSwipeLeft / OnSwipeRight event.
